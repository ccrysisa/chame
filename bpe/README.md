# LLM Tokenizer (BPE)

Using [BPE](https://en.wikipedia.org/wiki/Byte_pair_encoding) algorithm to implement/recreate a LLM tokenizer.

## Quick start

```sh
$ make
$ ./bpe
```

Generate grammer graph:

```sh
$ dot -Tpng rules.dot > rules.png
```
